EBS Integration.

The Discount Calculator Application is designed to effortlessly compute discount percentages based on changes in price, as well as determine the discounted price from a given discount percentage. This user-friendly tool caters to both consumers looking to understand their savings and retailers aiming to set competitive prices.

01

Install module/application.

02

Once the application or module has been installed, open it and fill out the integration requirements.

03

After filling out the form, press the "Check Connection" button.
If a connection is made, the stage moves to "Validate", else it says "Validation Error".

04

After the connection is established successfully, enter the customer's ( Buyer ) or company's ( Seller ) information.

05

The "EBS Response" page will display the response when payment is successful.

06

As well as for POS, you will be able to view the response on the bill receipt following a successful payment.

07

EBS Queue Jobs will display the response status, either as SUCCESS or ERROR.
The "Fail Status Description" box will display an error if the status is unsuccessful.

08

You can examine the ebs status of each of your invoices in tree view.

09

EBS responses for several invoices are available to you simultaneously.
